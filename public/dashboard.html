UX Analysis Overview

세션 라벨 분포와 인사이트를 한 번에 확인
Total Sessions
이번 조건으로 집계된 전체 세션 수입니다. 라벨 분포, 대표 유형, 인사이트 우선순위는 이 세션 집합을 기준으로 계산됩니다.
—
Primary Label
가장 많은 세션에 붙은 사용자 유형입니다. 지금 트래픽에서 어떤 행동 패턴이 가장 지배적인지 빠르게 보여줍니다.
—
High Priority Insights
현재 인사이트 중 우선순위가 `high`로 분류된 개수입니다. 비중이 크거나 마찰 강도가 높은 문제를 먼저 보게 해줍니다.
—
Insight Provider
인사이트를 만든 엔진입니다. `fallback`이면 규칙 기반 템플릿, `openai`면 실제 LLM 결과를 뜻합니다.
—
Label Distribution
사용자 유형별 세션 수와 비중입니다. 어떤 유형이 많아졌는지, 테스트 트래픽이 어디에 집중되는지 확인할 수 있습니다.
